About Me

My first introduction to Programming was in highschool. I wanted to try something different other than art. Didn't go so well. Though, later in my College years, I remember discovering the browser inspector tool and wondering what all that code was! I was interested in learning how to code, but, I couldn't find a good guide.

Fast foward many years, I built my own PC, later studied for the A+ exam, then learned to use the command line, which led me back to web development. It was the connection of writing HTML and CSS and visually seeing the immediate output that had me hooked!

After learning the basics, many sources say to start building projects. And it's true! I've been building projects from Frontend Mentor and Scrimba.

I'm currently learning how powerful templating languages are with static site generators, specifically 11ty with Netlify and Netlify CMS.
