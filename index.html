About Me

My first introduction to programming was in highschool. I wanted to try something different other than art. I Didn't quite get the hang of it then. Though, later in my college years, I remember discovering the browser inspector tool and wondered what all that code was! I spent some time then learning a little bit of HTML.

Many years later, I built my own PC, later studied for the CompTIA A+ exam, learned to use the command line, which led me back to web development, and it reminded me of typing out html tags. Once I discovered CSS though, and seeing visually the output, that had me hooked!

After learning the basics, many sources say to start building projects. And it's true! I've been building projects from Frontend Mentor, Scrimba, and Udemy to name a few.

I'm currently learning how powerful templating languages are with static site generators, specifically 11ty with Netlify and Netlify CMS. On the server side I'm also learning Wordpress development with PHP.
