About Me

My first introduction to programming was in highschool. I wanted to try something different other than art. I Didn't quite get the hang of it then. Though, later in my college years, I remember discovering the browser inspector tool and wondered what all that code was! I spent some time then learning a little bit of HTML.

Years later, I built my own PC, later studied for the CompTIA A+ exam, learned to use the command line, which reminded me of typing out HTML tags. That led the interest back to web development. Once I discovered CSS though, and seeing visually the output, that had me hooked!

I'm currently learning how to build MERN fullstack applications, and the Java programming language.

When I'm not programming I enjoy playing music(guitar, bass, sythesizers), watching movies, and auto repair.
